Interaction Contracts

  • DonationIntentInput creates card/crypto intent metadata.
  • DonorPortalService lists donations by donor or receipt token.
  • RssFeedService renders public campaign timeline entries for donations, GitHub publishing, and Hugging Face publishing.
  • RefundService creates and tracks Stripe-only refund batches, net of Stripe fees.

MVP Scope vs Out of Scope

In scope: donations, milestones, donor portal, RSS, Stripe refund-all workflow via sherifcherfa@gmail.com within 14 days.

Out of scope: admin analytics depth, multi-campaign support, advanced KYC tiers.

Convex + Keys

This model uses Convex schema + TypeScript contracts in this repo. Local credentials are in .env.local and ignored by git.

Webhook Runtime Wiring

Convex HTTP route: POST /stripe-webhook
Signature validation: Stripe v1 HMAC using STRIPE_WEBHOOK_SECRET
Idempotency key: stripeWebhookEvents.eventId
