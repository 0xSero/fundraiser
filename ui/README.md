# Fundraiser UI Prototype

Simple static UI prototype for the crowdfunding data model.

## Files
- `index.html`: page structure and section shells
- `styles.css`: minimal responsive styling
- `app.js`: mock data rendering and simple interactions

## Run
Open `index.html` directly in a browser:

```bash
open /Users/sero/ai/fundraiser/ui/index.html
```

No build step or dependencies are required.
